# Wallpaper SMTC Music

这是一个 Wallpaper Engine 音乐壁纸。

## 使用

- 默认可显示 Wallpaper Engine 官方媒体接口提供的歌名、歌手、封面和播放状态。
- 需要媒体源切换、播放控制、音量控制或更准确进度时，启动 `bridge/WallpaperMusicBridge.exe`。
- Bridge 是本机托盘程序，可在托盘菜单中启动服务、停止服务、重启服务或设置开机启动。

## 设置

常用设置在 Wallpaper Engine 右侧设置面板中：

- 播放器：启用播放器、默认显示、无音源是否隐藏、播放器样式。
- 歌词：启用歌词、在线歌词查询。
- 背景：封面背景、自定义壁纸、模糊强度、暗化效果。
- 媒体源：低优先级媒体源、屏蔽媒体源。

## 隐私提醒

启用在线歌词查询后，壁纸或 Bridge 可能会向第三方歌词服务发送歌名、歌手、专辑、媒体源和时长等信息。可以在设置中关闭“在线歌词查询”。

## Bridge 地址

Bridge 只在本机提供服务：

```text
http://127.0.0.1:18768
```

它不是外网服务，不需要管理员权限。
