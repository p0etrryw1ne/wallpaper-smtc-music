# Wallpaper Engine 官方媒体接口验收

## 目的

验证真实 Wallpaper Engine 媒体回调数据，并确认重构后的壁纸在未启动 Bridge 时也能工作。

## 调试入口

使用：

```text
index.html?debug=media
```

本地预览可用：

```text
http://127.0.0.1:8010/?debug=media
```

只有带上 `debug=media` 时才会显示调试面板。

- 使用打包后的 `release/we-smtc` 导入 Wallpaper Engine。
